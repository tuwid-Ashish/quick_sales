# Quick Sales - Gardening for Kids - Style Guide

This document outlines the new design system for the Quick Sales website, focusing on creating a fun, engaging, and kid-friendly experience.

## 1. Color Palette

The color palette is designed to be vibrant, energetic, and appealing to children.

- **Primary:** `#FFC107` (Amber) - A bright, cheerful yellow for primary calls-to-action, highlights, and branding.
- **Secondary:** `#4CAF50` (Green) - A natural, earthy green that reflects the gardening theme. Used for secondary buttons and accents.
- **Accent:** `#2196F3` (Blue) - A friendly blue for links, icons, and interactive elements.
- **Background:** `#F5F5F5` (Light Gray) - A soft, neutral background that makes the content pop.
- **Text:** `#333333` (Dark Gray) - A dark, readable color for all body text.
- **White:** `#FFFFFF` - For cards, modals, and other elevated surfaces.

## 2. Typography

The typography will be playful yet highly readable. We will use Google Fonts.

- **Headings:** `Nunito`, Bold, 700 - A rounded, friendly font for all headings.
- **Body:** `Nunito`, Regular, 400 - The same font for body text to maintain consistency and readability.

### Font Sizes
- `h1`: 3rem (48px)
- `h2`: 2.25rem (36px)
- `h3`: 1.5rem (24px)
- `p`: 1rem (16px)

## 3. Iconography

We will use a set of simple, rounded, and fun icons. The `lucide-react` library already in use provides a good base, and we will select icons that fit the playful theme.

- **Style:** Rounded, fill-style where possible.
- **Examples:** `Sprout`, `Flower`, `ShoppingCart`, `User`

## 4. Component Styling

Components will be updated to reflect the new, softer aesthetic.

- **Border Radius:** `8px` for all components (buttons, cards, inputs) to create a soft, rounded feel.
- **Shadows:** Soft, subtle shadows to create depth. `box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);`
- **Buttons:**
    - Solid background colors (Primary/Secondary).
    - White text.
    - Transitions for hover effects (slight brightness change).
- **Cards:**
    - White background.
    - Rounded corners.
    - Soft shadow.
    - Hover effect: slightly larger shadow or a subtle lift.

## 5. Layout

The layout will be clean and spacious, with plenty of white space to avoid feeling cluttered.

- **Spacing:** Use a consistent spacing scale (e.g., multiples of 8px) for margins and padding.
- **Grid:** A responsive grid system will be used for product listings and other content.

This style guide will be the foundation for all UI changes in Phase 3.
